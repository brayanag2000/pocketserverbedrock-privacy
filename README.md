# pocketserverbedrock-privacy
Privacy Policy for the PocketServerBedrock Android application.

# Política de Privacidad – PocketServerBedrock

PocketServerBedrock es una aplicación diseñada para permitir a los usuarios crear y ejecutar un servidor PocketMine-MP de forma local directamente desde su dispositivo Android.

## Recopilación de datos
La aplicación no recopila, almacena ni comparte información personal de los usuarios.  
No se solicitan datos como nombre, correo electrónico, ubicación, contactos ni información del dispositivo con fines externos.

## Uso de permisos
La aplicación puede solicitar permisos de almacenamiento únicamente para permitir el funcionamiento local del servidor, la gestión de archivos del sistema y la instalación de plugins importados por el usuario.  
Estos datos se procesan únicamente en el dispositivo y no se transmiten a servidores externos.

## Datos de terceros
PocketServerBedrock no utiliza servicios de terceros para recopilar datos ni integra sistemas de seguimiento, análisis o publicidad.

## Seguridad
Toda la información utilizada por la aplicación permanece en el dispositivo del usuario.  
La aplicación no establece conexiones remotas para transmitir datos personales.

## Menores de edad
La aplicación no está dirigida específicamente a menores de 13 años y no recopila información personal de niños.

## Cambios en esta política
Esta política puede actualizarse en futuras versiones para reflejar mejoras o cambios en la aplicación.

## Contacto
Si tienes preguntas sobre esta Política de Privacidad, puedes contactar al desarrollador a través de la información proporcionada en Google Play.
